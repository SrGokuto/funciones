¿Qué es una función?

Una función es un bloque o conjunto de codigo que se ejecutará al llamarlo.
Tomemos por ejemplo el siguiente codigo:

function saludar() {

lorem ipsumconsole.log("Hola! Este mensaje se ha enviado desde una función");

}

La función se llama saludar, y su funcionamiento es bastante simple: muestra un mensaje, en este caso, un saludo en consola. Sin embargo, al ejecutar dicho codigo, no aparecerá nada en consola, y eso es porque solo la declaramos, aún no hemos llamado/ejecutado la función!

Para esto, lo que hacemos es poner el nombre de la función en nuestro codigo de la siguiente forma:

saludar();

Con esos simples pasos, ya tenemos una función sencilla que envía un mensaje en consola!

Sin embargo las funciones pueden ser mucho más poderosas, pues estas pueden recibir datos para procesarlos de una forma u otra.

Uno de mis ejemplos favoritos es el de simplificar el mostrar cosas en consola:

function escribir(texto) {

lorem ipsumconsole.log(texto);

}

Es bastante sencillo de entender, y al mismo tiempo te mustra el poder que pueden llegar a tener las funciones si las utilizas de forma correcta.

Lo que hace es recibir los datos (Pueden ser texto o números, pues lo unico que hace es mostrarlos en consola) y se los manda a un console.log para mostrarlos en consola. La forma correcta de llamar a esta función es la siguiente:

escribir("Hola! este mensaje se muestra desde una función con parametros!");

Y tan simple como eso, se mostrará en consola!

Existen muchas otras formas de utilizar las funciones, por ejemplo para obtener un resultado de alguna operación, necesitas retornar el resultado:

function suma(a, b) {

lorem ipsumreturn a + b;

}

La función lo que hace es recibir dos valores, en este caso a y b, luego de eso, los suma, y procede a retornar el valor de sumar a + b.
para ejecutar dicha función, la llamamos de la siguiente forma:

let operacion = suma(1, 3);

console.log(operacion);

Y ya estaría, tan simple como eso, la función se ejecutará, sumará a + b, que en este caso sería 1 + 3, y mostrará el resultado en consola!

Tambien existen la llamadas "Funciones de flecha", las cuales sirven para crear funciones que se ejecutan al momento de declararlas, sirven para hacer operaciones algo más sencillas, pues cuando se empiezan a hacer más complejas, volverás a utilizar parametros como el return, los corchetes y demás
Las funciones de tipo expresión son algo más complejas, pues se pueden usar, por ejemplo, para declarar varias funciones en un array, luego puedes acceder a cada elemento del array, con lo que ejecuta cada función por separado

Los parametros son los datos que le das a la función, si es que los necesita.
Un valor de retorno puede ser cualquier valor aceptado por JavaScript, como puede ser un booleano, un string, un integer, un float, un double, un array, un object, etc

Al trabajar con funciones se recomienda seguir unas practicas, la más importante siendo el nombrar las funciones de forma descriptiva, para saber que hacer cada función

Se considera que una función es muy larga cuando tiene, en promedio, unas 35 lineas o más
